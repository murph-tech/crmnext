# ⚠️ Document Format Locked

รูปแบบเอกสารทั้ง 3 ประเภทต่อไปนี้ **ถูกล็อคแล้ว** และ **ห้ามแก้ไข** อีก:

## เอกสารที่ล็อค
1. **Quotation (ใบเสนอราคา)** - `/src/app/quotations/[id]/page.tsx`
2. **Invoice (ใบกำกับภาษี/ใบวางบิล)** - `/src/app/invoices/[id]/page.tsx`
3. **Receipt (ใบเสร็จรับเงิน)** - `/src/app/receipts/[id]/page.tsx`

## สิ่งที่ห้ามแก้ไข
- Layout และโครงสร้างของเอกสาร
- ขนาดและตำแหน่งของ text
- สีและ styling
- ช่องลายเซ็นและวันที่
- ตารางรายการสินค้าและสรุปยอดเงิน
- Header และ Footer

## สิ่งที่สามารถแก้ไขได้
- แก้ไข bug หรือ error ที่ทำให้ระบบไม่ทำงาน
- เพิ่ม logic ใหม่ที่ไม่กระทบต่อรูปแบบการแสดงผล
- แก้ไข Backend API
